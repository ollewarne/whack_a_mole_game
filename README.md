# Whack-A-mole

This is a simple game of Whack-A-mole.  
To play the game simply visit [THIS PAGE](https://ollewarne.github.io/whack_a_mole_game/)

## How to play?

After you have started the game with the start button you will get 60 seconds to click on as many moles as you can.  
The moles with pop up every second and stay for a duration between .5 and .9 seconds.  
If you click on a square with a mole inside you will get one point.  
If you don't click on the mole or click on the wrong square, you will get one miss.
